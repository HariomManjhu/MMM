#  MMM - Meal Macro Manager

**MMM (Meal Macro Manager)** is a web-based nutrition tracking app designed for mess-based meal systems, helping users log their meals, track calories and macros (protein, carbs, fat), and work toward fitness goals like cutting, maintaining, or gaining muscle.

---

##  Features

- User registration and secure login
- Track meals by day and type (breakfast, lunch, dinner)
- Input portion sizes (e.g., 1.5 servings of Poha, 0.8 of Upma)
- Automatic calculation of calories and macronutrients
- Dashboard for daily nutrient summary
- Track weight changes to estimate metabolism
- Set and adjust fitness goals
- Update physical stats (height, weight, activity level)
- 7-day repeating menu with 21 editable meals

---

##  Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: EJS Templates, Bootstrap 5
- **Database**: MySQL
- **Authentication**: bcrypt (password hashing), express-session
- **Templating**: EJS for dynamic HTML rendering
